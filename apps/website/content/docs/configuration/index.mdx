You can customize the configuration of your `xmcp` app by creating a `xmcp.config.ts` file.

```typescript
const config: XmcpConfig = {
  http: true, // builds your app with the HTTP transport
  stdio: true, // builds your app with the STDIO transport
};

export default config;
```

## Configuration Options

- [Custom directories](/docs/configuration/custom-directories) - Configure custom paths for tools, prompts, and resources
- [HTTP transport](/docs/configuration/http-transport) - Customize the HTTP server configuration
- [Webpack](/docs/configuration/webpack) - Extend the webpack configuration
- [Experimental features](/docs/configuration/experimental) - Enable experimental features
